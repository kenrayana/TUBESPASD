# 🧠 MindFlow
### AI-Powered Student Mental Wellness Dashboard

MindFlow is an AI-powered dashboard designed to help students monitor their mental wellness and predict stress levels based on daily academic and lifestyle factors. The application integrates Machine Learning with an interactive Streamlit interface to provide personalized insights and recommendations.

---

## 📖 Project Overview

University students often experience stress caused by academic workload, sleep quality, extracurricular activities, and overall performance.

MindFlow aims to assist students by:

- 📊 Monitoring daily wellness through a simple check-in.
- 🤖 Predicting stress levels using a Machine Learning model.
- 💡 Providing AI-generated recommendations.
- 📈 Visualizing readiness and productivity forecasts.

---

## ✨ Features

### 📝 Student Check-In
Users provide daily information including:

- 😴 Sleep Quality (1–10)
- 📚 Academic Performance (1–10)
- 📖 Study Load (Light / Medium / Heavy)
- 🏃 Extracurricular Activity (1–10)

---

### 🤖 Machine Learning Prediction

The application predicts:

- Stress Level
- Daily Readiness Score
- Energy Level
- Productivity Status
- Best Productivity Time

---

### 📊 Dashboard

The dashboard includes:

- Daily Readiness Card
- Current Status
- Productivity Indicator
- Productivity Forecast
- Forecast Pattern Chart
- AI Insight & Recommendations

---

### 🎨 Modern User Interface

- Responsive dashboard
- Dark & Light mode support
- Interactive radio-button input
- Modern cards and gradients
- User-friendly layout

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/kenrayana/TUBESPASD.git
```

Move into the project folder

```bash
cd TUBESPASD
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
